A **powerful Python-based Cybersecurity & OSINT Toolkit** for reconnaissance, information gathering, and ethical hacking practice.  
This toolkit combines multiple tools including **IP Tracker, Port Scanner, Whois Lookup, Subdomain Finder, Email Harvester, Mobile Number Trace, and Instagram OSINT**.

---

## ⚡ Features

- **IP Tracker:** Locate any IP address with geolocation info.
- **Port Scanner:** Scan open ports of hosts.
- **Whois Lookup:** Get domain registration details.
- **Subdomain Finder:** Enumerate subdomains of a target domain.
- **Email Harvester:** Extract email addresses from websites.
- **Mobile Number Trace:** Trace mobile number location (publicly available info only).
- **Instagram OSINT:** Collect public profile info, download profile pics, generate reports.
